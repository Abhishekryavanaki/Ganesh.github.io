CSS Grid Layout Crash Course

Example-1

To display a CSS Grid, just use display:grid;

.wrapper{
display:grid;
grid-template-columns:70% 30%;
grid-gap: 1em;
}

This shows 4 boxes having a width of 70% and the white boxes a width of 30%.

These two lines are the same values, which can be replaced with grid-gap .Grid-gap allow for each section of the grid to have spacing between them.

grid-column-gap:1em;
grid-row-gap:1em;

Example 2

The fr unit allows you to set the size of a track as a fraction. repeat() definition contains repeating parts.

Another

Grid can

be nested

inside of

this

grid.

.example-2 .wrapper {
grid-template-columns: repeat(3, 1fr);
grid-gap:2em;
}

The class wrapper is the main grid for this example and nested is the new grid inside of the first box.

grid-auto-rows sets the row height. This is a fixed number. If the row height exceeds the givern value, then the content will overflow.
The value minmax(100px, auto) allows there to be a minimial height of 100px and if it exceeds that height the content will auto fix to a height to fit the content.
